# Seed data for CI / first-run

Эта папка содержит снапшот сгенерированного `out/` (доки + каталоги действий), который используется:

- в CI/Release, чтобы собирать установщик без локальных файлов (`regallactions_export.txt`, `apples.txt`);
- при установке, чтобы `mldsl.exe` работал “из коробки”.

Важно:

- это реальный результат `python tools/build_all.py` с твоей версией экспорта;
- чтобы обновить seed, пересгенерируй `out/` и перезапиши `seed/out/`.

## Source inputs

В `seed/inputs/` хранятся важные исходники, которые долго восстанавливать:

- `regallactions_export.txt` (сырой экспорт действий из игры);
- `apples.txt` (опциональные apple-константы).

Эти файлы используются как fallback-источник, если локальные файлы пользователя недоступны.
