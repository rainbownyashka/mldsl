## Description
```
Округляет нецелое число до целого  
 за всеми правилами математики. 
  
 В сундуке: 
 Динамическая переменная -  
 переменная для установки. 
 Число - значение, которое  
 нужно округлить. 
 Число* - Число после запятой 
 Переключатель - Тип округления 
  
 *Необязательно 
  
 Пример: 
 Var.round(3.4) = 3 
 Var.round(3.5) = 4  
  
 Данное действие тебе доступно!
```
