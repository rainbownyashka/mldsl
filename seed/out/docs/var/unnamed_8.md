```

 на 1 или другое число. 
 Не работает с нецелыми числами, 
 используйте "Установить (+)". 
  
 В сундуке: 
 Динамическая переменная - переменная 
 для прибавление 
 Число* - Число для увеличения 
 *Необязательно, по стандарту 1 
  
 Пример: 
 Var += 5 
 (Var = Var + 5) 
  
 Данное действие тебе доступно!
```
