```

 Если синус угла равен отношению противолежащего катета к гипотенузе 
 прямоугольного треугольника, то арксинус угла дает сам угол. 
  
 В сундуке: 
 Динамическая переменная - для установки результата 
 Число - Число для преобразования в арксинус 
  
 Пример: 
 Var = asin(0) = 0.0 
 Var = asin(0.5) ≈ 0.52 
  
 Данное действие тебе доступно!
```
