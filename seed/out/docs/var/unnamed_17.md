## Description
```
Возвращает арккосинус в радианах. 
 Если косинус угла равен отношению противолежащего катета к гипотенузе  
 прямоугольного треугольника, то арккосинус угла позволяет найти сам угол. 
  
 В сундуке: 
 Динамическая переменная - для установки результата 
 Число - Число для преобразования в арккосинус 
  
 Пример: 
 Var = acos(0) ≈ 1.57 
 Var = acos(0.5) ≈ 1.05 
  
 Данное действие тебе доступно!
```
