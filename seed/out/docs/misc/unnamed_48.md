## Description
```
Выполняет код, когда игрок взаимодействует с блоком 
  
 Дополнительная информация: 
 > В некоторых случаях событие может 
 сработать два раза. Одна из причин может 
 быть в том, что было взаимодействие двух рук, 
 правой и левой. Используйте проверку "Тип слота снаряжения 
 равен" [Основная рука] для исправления проблемы. 
 > В некоторых случаях событие может не сработать,  
 например, клик по блоку в режиме приключений 
 > Событие можно отменить 
  
 Примеры срабатывания: 
 > Игрок кликнул по воздуху, держа предмет в руке 
 > Игрок открыл сундук 
 > Игрок растоптал грядку 
  
 Данное событие тебе доступно! ;)
```
