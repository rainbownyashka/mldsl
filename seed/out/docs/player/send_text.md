## Description
```
Отправляет сообщение в чат 
 указанным игрокам. 
  
 В сундуке: 
 Текстовый(ые) компонент(ы) - Текст для отправки 
 Переключатель - Разделитель между 
 сообщениями 
  
 Дополнительная информация: 
 > Числа, местоположения и 
 другие значения будут  
 преобразованы в текст. 
  
 Данное действие тебе доступно!
```
