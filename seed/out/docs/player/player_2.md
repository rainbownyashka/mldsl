## Description
```
Делает визуально игрока лежачим 
 или отображает ему меню лежания 
  
 В сундуке: 
 Местоположение - Местоположение 
 начала кровати (без подушки) 
 Переключатель - Тип отображения 
  
 Дополнительная информация: 
 > Если на местоположении не  
 будет кровати, то игрок станет невидимым, 
 если его попытались отобразить лежачим 
 > Игроки которые зайдут 
  в игру не увидят его лежачим. 
  
 Данное действие тебе доступно!
```
