## Description
```
Загружает сохранённый инвентарь игрока в игре. 
  
 Дополнительная информация: 
 > Если у игрока нет сохранённого инвентаря,  
 то загружает пустой инвентарь. 
 > Самый простой способ использования этого  
 "Событие входа -> Загрузить инвентарь" 
   
 Данное действие тебе доступно!
```
