## Description
```
Сравнивает числа проверками по типу 
 "больше", "меньше", "больше или равно", 
 "меньше или равно" 
  
 В сундуке: 
 Число - Число, которое нужно сравнить 
 Число* - Число, которое должно быть меньше 
 Число* - Число, которое должно быть больше 
 2 переключателя - Тип сравнения "Больше" и "Меньше" 
 *Необязательно 
  
 Данное действие тебе доступно!
```
